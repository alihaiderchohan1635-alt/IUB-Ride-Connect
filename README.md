# IUB-Ride-Connect
IUB Ride Connect — An online ride booking platform for students and staff of The Islamia University of Bahawalpur, enabling real-time ride matching with verified university drivers.
IUB Ride Connect is an online ride booking system built for the students and staff of The Islamia University of Bahawalpur (IUB). It solves daily commute challenges within and across IUB campuses by offering real-time ride matching, verified student/staff drivers, fare estimates, and multi-campus coverage — all through a simple, low-cost, and secure platform.
Built with HTML, CSS, JavaScript (frontend) and PHP, MySQL (backend), hosted locally via XAMPP.
